

# Bhagavad Gita RAG Assistant 🕉️

A Voice-Enabled AI Chatbot for the Bhagavad Gita, capable of answering questions based on the sacred text, providing verses, and supporting multilingual voice interaction (English, Hindi, Kannada, Sanskrit).

## Features
- **RAG Architecture**: Accurately answers from the Bhagavad Gita.
- **Voice Chat**: Speak your questions and listen to the answers.
- **Multilingual**: Supports English, Hindi, Kannada, and Sanskrit.
- **Verse of the Day**: Get daily inspiration.
- **Emotion Support**: Ask for help based on your feelings (e.g., "I am angry").

## Tech Stack
- **Frontend**: Streamlit
- **LLM**: LLaMA 3.1 (via Groq)
- **Vector DB**: Qdrant
- **Orchestration**: LangGraph
- **Voice**: Whisper (STT) + gTTS (TTS)
